# 指标定义

## 核心指标

### 活跃率（周活率）

- **定义**：指定周期内，至少使用过一次任意能力的用户数 / 总用户数（已排除离职）
- **公式**：`活跃用户数 / 有效总用户数`
- **注意**：活跃 = 该周期内任意能力使用次数 > 0

### 渗透率

- **定义**：使用过某项能力的用户数 / 总用户数
- **公式**：`使用过该能力的用户数 / 有效总用户数`

### 人均使用能力数

- **定义**：用户在整个分析周期内，使用过的不同能力类别数
- **公式**：对每个用户，统计各能力列 > 0 的列数，再取均值

### 环比变化

- **定义**：当前周期指标值与上一周期指标值的差异百分比
- **公式**：`(当前值 - 上一周期值) / 上一周期值 * 100%`
- **注意**：若上一周期值为 0，显示 "N/A"

## 辅助指标

| 指标 | 说明 |
|------|------|
| 总使用次数 | 各能力使用次数之和 |
| 活跃周数 | 用户在分析周期内处于活跃状态的周数 |
| 周使用次数 | 单周内各能力使用次数之和 |
