# 字段说明

## 使用明细表标准字段

| 列名 | 类型 | 维度 | 说明 | 示例 |
|------|------|------|------|------|
| 姓名 | string | 人员标识 | 用户唯一标识（人名） | 张三 |

| 部门 | string | 人员身份 | 所属部门 | 销售一部 / 研发中心 |
| 角色 | string | 人员身份 | 一级角色分类 | Sales / Sales Manager |

## 维度分类

### 人员维度（6个）
- `部门` — 所属部门（如销售一部、研发中心）
- `角色` — 一级角色（如 Sales、Sales Manager）
- `产品线` — 产品线/事业部
- `大区` — 区域划分
- `用户角色` — 二级角色细分
- `层级` — 职级/管理层级（如 L1、L2、M1）

### 时间维度（1个，支持3种粒度）
- `财周` — 财务周序号，作为时间聚合键
  - 支持粒度：周（week，默认）、月（month）、季度（quarter）

### 能力维度（6个）
- 能力一、能力二、能力三、能力四、能力五、能力六

## 默认聚合参数

- 身份维度默认：`部门,角色,产品线,大区,用户角色,层级`
- 时间维度默认：`财周`
- 能力列自动检测规则：列名含"使用次数"即视为能力列
